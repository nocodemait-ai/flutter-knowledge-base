This package is one of the software components in the the overall [Copenhagen Research Platform (CARP)](https://carp.cachet.dk).

This package contains the [CARP Core](https://github.com/carp-dk/carp.core-kotlin?tab=readme-ov-file#carp-core-framework) domain model as implemented in Dart. It contains the domain model used in the [carp_mobile_sensing](https://pub.dev/packages/carp_mobile_sensing) framework and all of its supporting libraries, packages, etc.

This package does nothing on its own.

## Documentation

* For an overview of CARP Mobile Sensing (CAMS), see the [CARP Mobile Sensing](https://carp.cachet.dk/cams/) homepage. Also see the [carp_mobile_sensing](https://pub.dev/packages/carp_mobile_sensing) framework on pub.dev, as well as the [API documentation](https://pub.dev/documentation/carp_mobile_sensing/latest/).

* For an overview of all CAMS packages, see [CARP Mobile Sensing in Flutter](https://github.com/carp-dk/carp.sensing-flutter).

* For documentation on how to use CAMS, see the [documentation](https://docs.carp.dk/carp-mobile-sensing/).

* For a thorough description of the CARP Core domain model, see the [carp.core-kotlin](https://github.com/carp-dk/carp.core-kotlin?tab=readme-ov-file#table-of-contents) documentation. The [`example.dart`](https://pub.dev/packages/carp_core/example) of this package illustrates how the [Kotlin examples](https://github.com/carp-dk/carp.core-kotlin#example) are implemented using Dart.
